# INSTRUCITONS

## TO RUN THE APP

```bash
git clone https://github.com/HarshaVardhanNakkina/amdocs-mysql
cd amdocs-sql
npm install

# to start the development server
npm run dev:windows # for windows
npm run dev:unix # for unix-like systems

# to start the production sever
npm run start
```

## TO CONNECT TO THE DATABASE

- create a `.env` in the root folder of the application
- copy the values from the `.env.example`
- fill the details accrodingly
